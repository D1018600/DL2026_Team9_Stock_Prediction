# **股票收盤價預測期末專題**



### 

### **專題目標**

本專題的核心目標在於探討並比較深度學習中各種循環神經網路架構與自注意力機制架構在時間序列預測上的表現。

我們採用台灣股市指標性個股——台達電（2308）的歷史交易數據，利用過去連續n天（本專案設定滑動視窗 `WINDOW = 20`，即前 19 天的歷史特徵資料）的複合指標，來預測第 n+1 天的收盤價。



評估指標

為了客觀衡量各模型的預測準確度與實際應用價值，本專案採用以下雙重指標：

\- MSE (Mean Squared Error，均方誤差)：作為模型訓練時的損失函數，用以評估模型整體的擬合能力。

\- MAE (Mean Absolute Error，平均絕對誤差)：用於計算預測股價與真實股價之間的平均絕對金額偏差，提供具實際經濟意義的直觀評估。

### **模型架構設計**



本專案基於 \*\*PyTorch\*\* 深度學習框架，針對時間序列預測任務實現了四種具代表性的網路架構。所有模型的輸入維度皆為 5（包含 `open`, `close`, `low`, `high`, `volume`），隱藏層維度（Hidden Size / d\_model）皆統一設定為 32 以確保對比的公平性：



* **LSTM**

&#x20; 透過遺忘閘、輸入閘與輸出閘的設計，能有效減緩傳統 RNN 的梯度消失問題，專長於捕捉長週期的時間序列依賴性。

* **BiLSTM**

&#x20; 雙向長短期記憶網路。同時架構正向與逆向兩個獨立的 LSTM 序列，試圖從時間軸的前後雙向脈絡中擷取更豐富的特徵資訊，並在輸出層融合兩者的隱藏狀態進行迴歸預測。

* **GRU**

&#x20; 將 LSTM 的架構簡化為更新閘與重置閘。因其結構精簡、參數較少，在中小規模的時間序列數據上通常具備更快的收斂速度，且預測表現往往能與 LSTM 旗鼓相當。

* **Transformer**

&#x20; 捨棄了傳統的循環結構，完全基於自注意力機制。它能夠同時平行處理整個時間序列滑動視窗內的資訊，自動計算不同日期之間特徵的關聯權重，在捕捉複雜的長期動態趨勢上展現出強大的潛力。

### **使用方法與執行流程**



1\. 環境準備

&emsp;&emsp;執行環境已安裝以下 Python 核心資料科學與深度學習套件：

&emsp;&emsp;pip install numpy pandas matplotlib torch scikit-learn tqdm onnx onnxscript


2\. 核心執行步驟

&emsp;&emsp;請依照 Notebook 中的儲存格順序依次執行：


&emsp;&emsp;Step 1：資料載入與排序

&emsp;&emsp;&emsp;&emsp;程式會自動下載 2308.csv 數據，並嚴格依據時間軸進行由遠到近的升序排列。


&emsp;&emsp;Step 2：資料集切分與標準化（防範資料洩漏）

&emsp;&emsp;&emsp;&emsp;將前 85% 的歷史資料劃分為訓練集，後 15% 劃分為驗證集。使用 sklearn 的 StandardScaler 

&emsp;&emsp;&emsp;&emsp;對 \['open', 'close', 'low', 'high', 'volume'] 進行標準化。注意：僅使用訓練集擬合統

&emsp;&emsp;&emsp;&emsp;計量，驗證集僅進行 transform，以避免資料洩漏。


&emsp;&emsp;Step 3：構建滑動視窗

&emsp;&emsp;&emsp;&emsp;設定 WINDOW = 20。每次抽取連續 19 天的 5 個標準化特徵作為網路輸入（X），並

&emsp;&emsp;將第 20 天的 close作為預測標籤Y）。


&emsp;&emsp;Step 4：模型訓練與優化

&emsp;&emsp;&emsp;&emsp;優化器：Adam (Learning Rate = 1e-3)

&emsp;&emsp;&emsp;&emsp;損失函數：nn.MSELoss()

&emsp;&emsp;&emsp;&emsp;訓練輪數：30 Epochs

&emsp;&emsp;&emsp;&emsp;程式在訓練過程中，每隔 5 個 Epoch 會自動輸出當前的 Train Loss 與 Valid Loss，方便追蹤模型的收斂狀況。


&emsp;&emsp;Step 5：反標準化與指標評估

&emsp;&emsp;&emsp;&emsp;當模型完成預測後，預測值會經由 StandardScaler.inverse\_transform 還原成真實的台幣股價金額，進而計算出

&emsp;&emsp;&emsp;&emsp;最終的Valid MSE 與 MAE，並繪製出真實股價與各模型預測股價的對比折線圖。


&emsp;&emsp;Step 6：模型自動匯出

&emsp;&emsp;&emsp;&emsp;程式最後會調用 torch.onnx.export，將四個訓練完成的 PyTorch 模型即時轉換為 .onnx 格式檔案並下載儲存。

### **實驗結果深入分析與討論**



1\. 循環神經網路的優勢：



在本實驗中，GRU 取得了最優的預測成果。這反映出在中小規模的金融時間序列資料集中，輕量化的網路架構擁有更強的泛化能力。而雙向的 BiLSTM 在此任務中表現並未超越單向 LSTM，主因是股票預測屬於單步前向預測，未來資訊的反向傳遞對當前狀態的修正效果有限。



2\. Transformer 模型的過擬合現象：



若對比訓練集數據，Transformer 在訓練集上的擬合效果極佳，但在驗證集上的 MSE 卻放大至 444.611。這是非常典型的過擬合現象。金融市場充滿雜訊，而 Transformer 擁有龐大的自注意力機制與複雜參數，在單一股票僅數年的歷史日 K 線數據下，容易「硬背」訓練集的雜訊，導致在未見過的驗證集上泛化能力下降。



3\. 專題結論：



本專題成功驗證了不同深度學習架構在金融時間序列上的適用性與侷限性。 Transformer 自注意力機制雖然擅長捕捉長距離的複雜特徵關聯，但非常依賴海量數據；針對特定個股的常規預測任務，傳統且輕量化的循環神經網路依然是現階段兼具效率與精準度的首選。
